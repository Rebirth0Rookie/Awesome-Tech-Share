## Installation and Setup

```shell
conda create -f environment.yml

git clone https://github.com/KinnariyaMamaTanha/mkdocs-statistics-plugin
pip install ./mkdocs-statistics-plugin
rm -rf mkdocs-statistics-plugin

pip cache purge
```

```shell
mkdocs serve
```
